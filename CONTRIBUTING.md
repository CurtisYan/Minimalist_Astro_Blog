# Contributing

Thanks for improving this template.

## Bilingual Documentation

The README files are split by language, and the two guide articles are bilingual inside the blog:

- `README.md` and `README.en.md`
- `src/content/posts/tech/2026-06-23-template-features.md`
- `src/content/posts/tech/2026-06-24-template-usage.md`

When you change one language section, update the matching section in the same pull request.

## Before Opening a Pull Request

Run:

```bash
npm run build
```

Keep sample content generic. Do not add personal domains, private email addresses, tracking IDs, or production credentials to the template.
